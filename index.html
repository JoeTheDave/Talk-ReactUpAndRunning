render

required

should be kept pure

componentDidMount

Invoked once immediately after the DOM is initially rendered.

The DOM representation of the component can be found and acted up via ReactDOM.findDOMNode(component)

componentWillReceiveProps

Invoked when a component has it's props changed, but not the first time during the initial render.

componentWillMount

Invoked onced before DOM is initially rendered.

Used for setting initial state.

shouldComponentUpdate

Invoked before rendering when updating, not on intial render.

Used to tell react not to update when you know there will be no visual effect.

Return false from this method to cancel the update.

componentWillUpdate

Invoked immediately before rendering when new props or state are recevied. Not on initial rendering.

componentWillUpdate(object nextProps, object nextState) { }

componentDidUpdate

Invoked immediately after DOM is updated, not on the initial render.

componentDidUpdate(object prevProps, object prevState) { }

You can operate on the DOM with reactDOM.getDOMNode(this)

componentWillUnmount

Invoked immediately before a component is unmounted from the DOM.

Use to clean up any DOM manipulations or Timers created in componentDidMount or componentDidUpdate.
